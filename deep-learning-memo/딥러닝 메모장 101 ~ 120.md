```
조기 종료 (Early Stopping)

에폭을 30~50으로 크게 잡았을 때, Val Loss가 몇 에폭 동안 연속으로 낮아지지 않으면 학습을 스스로 끝내는 안정장치다.

[파이썬 코드]
patience = 5 # 참을성: 5에폭 동안 점수가 안 좋아지면 끝내겠다.
patience_counter = 0
best_val_loss = float('inf')

for epoch in range(epochs):
	# ...학습 및 검증 진행...
	
	# 이번 에폭의 val_loss가 최고 기록을 갱신했는지 확인
	if current_val_loss < best_val_loss:
		best_val_loss = current_val_loss
		patience_counter = 0 # 갱신했으니 초기화
		total.save(model.state_dict(), 'best_model.pth')
	else:
		patience_counter += 1 # 갱신 실패했으니 카운트 증가
	
	if patience_counter >= patience:
		break # 모델의 발전이 없으니 조기종료.
```

```
가중치 동결 및 해제 (Freeze / Unfreeze)

Pretrained 모델(ResNet 등)을 가져왔을 때, 앞부분 레이어들은 이미 똑똑해진 상태이다.
그래서 처음 몇에폭 동안은 앞부분 가중치를 잠그고(Freeze) 내가 새로 만든 뒷부분(model.fc)만 먼저 학습시킨 뒤, 나중에 자물쇠를 풀고(Unfreeze) 전체를 파인튜닝하는 테크닉.

[파이썬 코드]
# 1. 모델 로드 직후: 전체 가중치 동결 (Freeze)
model = models.resnet18(weights=models.ResNet18_Weights.DEFAULT)
for param in model.parameters():
	param.required_grad = False # 역전파 때 미분(학습) 금지

# 2단계: 맨 뒷부분 예측 레이어만 교체
model.fc = nn.Linear(model.fc.in_features, len(classes))

# 이 상태로 Epoch 1~2 정도 돌리면 앞에는 고정되고 fc만 빠르게 학습된다.

for param in model.parameters():
	param.requires_grad = True # 자물쇠 풀고 전체 다 같이 학습 시작
```

```
가중치 동결이 오히려 성능을 깎아먹는 이유

- 가중치 동결했을 때 (96~97%)
ResNet18의 뼈대를 그대로 얼려두면 이 모델은 "일반적인 세상의 모든 사물을 다 잘 알아보는 뇌" 상태로 고정됨.
그 상태에서 맨 마지막 레이어(model.fc)만 바꾸고 학습시키면 원래 가지고 있던 일반적인 시각 능력만 가지고 구별해야 한다.
하지만 ImageNet에는 갈변부분 같은 아주 미세한 특징이 깊게 학습되어 있지 않다.
즉, 뼈대가 과일의 디테일을 잡기엔 너무 둔감했다.

- 가중치 동결하지 않았을 때 (99.76%)
모든 가중치를 열어두고 학습(파인튜닝)시키면 ResNet18이 가진 수천만개의 가중치가 과일의 신선도를 위해 통째로 재설계된다.


실무에서 가중치 동결 여부를 결정하는 기준
(1) 가중치 동결이 유리할 때: 데이터가 너무 적어서 전체를 학습시키면 과적합이 일어나거나, 컴퓨터 사양이 너무 낮아서 빠른 학습이 필요할 때

(2) 전체 파인튜닝이 유리할 때: 지금처럼 데이터 수가 충분하고, 맞추고자 하는 목표가 기존 이미지넷 데이터와 성격이 조금 달라서 모델의 가중치들을 미세하게 조정해줄 필요가 있을 때


그렇다면.. 현업에서는 가중치 동결을 거의 하지 않는건가? 그렇지 않다.
현업의 데이터는 매우 불균형하고 더럽기 때문이다.
제조업 공장 라인에서 '정상 제품'은 수만 장씩 쏟아지지만, '불량 제품'은 엄청 드물다.

동결이 필요한 이유: 데이터가 수만장 있어도 데이터 불균형이 있으면 전체 가중치를 열어놓고 학습할 때 모델이 정상 데이터에만 과적합되는 문제가 발생할 수 있다.
이럴 때 기존의 사물의 형태를 잘 파악하도록 훈련된 백본 모델의 가중치를 동결해 두고, 뒤쪽 분류기(fc)만 영리하게 학습시켜서 모델의 기본 시각 능력이 망가지는 걸 방지한다.
```
